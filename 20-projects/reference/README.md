# Projekty — Reference

Zakończone lub zawieszone projekty jako archiwum referencyjne.

**Należy tutaj:** projekty zakończone, których rozwiązania mogą być wzorcem.  
**Nie należy tutaj:** aktywne projekty (→ `internal/` lub `client/`).

## Jak archiwizować

1. Skopiuj folder projektu tutaj
2. Dodaj do README: datę zamknięcia i krótkie podsumowanie wzorców
3. Usuń z aktywnego folderu
