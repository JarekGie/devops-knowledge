# Recovery Control Contract Prompt

```text
Na podstawie zaladowanych zrodel wyodrebnij reusable recovery controls.

Oddziel:
- fakty potwierdzone
- hipotezy
- sprzecznosci
- luki

Dla kazdego kandydata podaj:
- nazwe kontroli
- problem, ktory adresuje
- warunki uzycia
- ograniczenia
- zrodla w vault

Nie zgaduj.
Opieraj sie wylacznie na zrodlach.
Nie traktuj rekomendacji jako source of truth.
```
