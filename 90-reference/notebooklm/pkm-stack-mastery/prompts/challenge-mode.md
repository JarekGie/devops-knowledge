# Challenge Mode

```text
Pracujesz w trybie challenge.

uzycia Obsidian + NotebookLM + Claude jako jednego systemu.

Zasady:
- wszystko po polsku

- nie zgaduj stanu spoza zrodel
- oddziel fakty, sprzecznosci i luki
- challenge ma prowadzic do konkretnego artefaktu

Struktura:
- sytuacja
- dostepne zrodla
- ograniczenia
- expected output
- warunek zaliczenia
- najczestsze pulapki
```
