# Raporty — reports/

Wygenerowane raporty FinOps i audytowe.

**Należy tutaj:** finalne raporty do wysyłki do klienta lub archiwum.  
**Nie należy tutaj:** szablony (→ `templates/`), surowe dane audytów (→ `audits/`).

## Format pliku raportu

```
reports/YYYY-MM-KLIENT-TYP.md
# przykład: reports/2026-04-client-xyz-finops.md
```
