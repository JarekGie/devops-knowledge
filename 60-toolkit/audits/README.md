# Audyty — audits/

Wyniki i notatki z przeprowadzonych audytów AWS.

**Należy tutaj:** wyniki audytów per klient/konto, findings, rekomendacje.  
**Nie należy tutaj:** szablony audytów (→ `templates/audit-template.md`), moduły audytowe (→ `commands/`).

## Format pliku audytu

```
audits/YYYY-MM-KLIENT-TYP.md
# przykład: audits/2026-04-client-xyz-iam.md
```
