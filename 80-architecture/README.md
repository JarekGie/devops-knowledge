# 80 — Architecture

Decyzje architektoniczne, mapy systemów, zasady platformy.

**Należy tutaj:** ADR (Architecture Decision Records), mapy systemów, principy.  
**Nie należy tutaj:** runbooki (→ `40-runbooks/`), dokumentacja implementacyjna (→ projekty).

## Szybka nawigacja

- [[decision-log]] — historia decyzji architektonicznych
- [[system-maps]] — diagramy i mapy systemów
- [[platform-principles]] — zasady projektowania platformy
- [[integration-notes]] — integracje z zewnętrznymi systemami

## Jak dodawać decyzje

Skopiuj `templates/decision-template.md` i dodaj wpis do [[decision-log]].  
Alternatywnie: dopisz wpis bezpośrednio do `decision-log.md` używając formatu sekcji.
